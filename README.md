## Instalation

- Install dependencies [yarn](https://yarnpkg.com/lang/en/docs/install/), [node & npm](https://github.com/creationix/nvm#installation) in your local
- clone repository

```sh
cd web_dev
cp .env.example .env
yarn
yarn start
```
